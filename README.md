## 🛠️ Get started

To use drf-redesign, follow these simple steps:

Install the package:

```shell
pip install drf-redesign
```

Add `drf_redesign` to `INSTALLED_APPS` setting, before `rest_framework`

```python
# settings.py

INSTALLED_APPS = [
    # ...,
    'drf_redesign',
    'rest_framework',
    ...
]
```

That's it! You're ready to go. 😎
